GestPracticas - paquete demo

Contenido:
- Aplicacion Next.js compilada en modo standalone.
- Archivos estaticos necesarios para ejecucion.


Uso:
1. Restaura manualmente en PostgreSQL la base de datos que quieras usar en la demo:
   crea manualmente la bbdd, por ejemplo "gestpracticas_demo";
   restaura si quieres los datos de ejemplo de "gestpracticas_demo.backup";
2. Configura la variable DATABASE_URL en .env si hace falta.
3. Ejecuta: node server.js
